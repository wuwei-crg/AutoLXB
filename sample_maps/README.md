# Sample Navigation Maps

Pre-built navigation maps for trying out LXB without running the map builder yourself.

| File | App | Package | Built |
|------|-----|---------|-------|
| `bilibili.json` | Bilibili | `tv.danmaku.bili` | 2026-02-27 |
| `taobao.json` | Taobao | `com.taobao.taobao` | 2026-02-25 |

**Device:** Redmi Turbo 4 Pro, Android 16, 2772×1280

> Maps are device/version-specific. If the app has updated or your device differs significantly, some edges may fail to replay. Re-running the map builder is recommended for production use.
